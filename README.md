# Japanese Language Flashcards

System: React + TypeScript + Vite

# Setup

## Initial Setup
1. Clone repo
2. npm install

## Local development
1. npm run dev

This will run vite preview on local.  
Tip: 'o' will open the main page on your browser.

## Deploy
1. npm run build
2. npm run publish
3. Push to origin/master on Github. Github actions will then deploy the site onto Github Pages.
